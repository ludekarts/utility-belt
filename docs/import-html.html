importHtml(doneCallback)

Allows to fetch HTML content from another .html file and renders its content into given node. This helper works synchronously. After html is loaded container node fires htmlLoaded event.

Usage

  import { importHtml } from "utility-belt";
  ...

  \\importHtml()\\;

Now in the HTML you need to add data-import attribute in order to fetch data

  <div \\data-import\\="path/to/file.html"></div>

Props

Name Type Description
doneCallback Function Function called after all data are loaded
