deepOverride(target, source)

Allows to deep override target object with source config.


Helper always prefers source object over target, except the case when the source property is undefined.

Usage

  import { deepOverride } from "utility-belt";
  ...

  const \\target\\ = {
    user : {
      name: "John",
      age: 22,
      gender: "male",
      metadata: {
        ststus: "Admin",
      }
    }
  };

  const \\source\\ = {
    user: {
      name: "Marry",
      age: 21,
      gender: "female",
      metadata: {
        auth: true,
      }
    }
  };

  console.log(deepOverride(\\target\\, \\source\\));

  -\/* {
    user: {
      name: "Marry",
      age: 21,
      gender: "female",
      metadata: {
        auth: true,
        ststus: "Admin"
      }
    }
  } */\-

When overriding arrays in order to update specific values the structure of the array need to be preserved e.g.:

  deepOverride(
    [1, 2, \\3\\],
    [undefined, undefined, \\4\\]
  );
  -\--> [1, 2, 4]\-

  deepOverride(
    [1, 2, \\3\\],
    [\\4\\, \\5\\]
  );
  -\--> [4, 5, 3]\-

null is treated as a value. Below example of overriding whole array:

  deepOverride(
    [1, 2, 3],
    \\null\\
  );
  -\--> null\-

Props

Name Type Description
target Any An instance that values will be copied to
source Any An instance that values will be copied from
