hyphenate(phrase)

Remove Polish diacritics and hyphenate name. Usefull when creating URL slugs e.g. for articles.j

Usage

    import { hyphenate } from "utility-belt";
    ...

    console.log(
      hyphenate("\\Cześć wam wszystkim! - mówi John\\") -\// czesc-wam-wszystkim-mowi-john\-
    );

Props

Name Type Description
phrase String Phraser to be hyphenate.

fuzzySearch(characters, phrase)

Checks if given characters are present in the phrase.

Usage

    import { fuzzySearch } from "utility-belt";
    ...

    console.log(
      ["Love", "Dragon", "Boring", "World"].filter(word => fuzzySearch("\\on\\", word))
    );
    -\// ["Drag\\on\\", "B\\o\\ri\\n\\g"]\-

Props

Name Type Description
characters String Characters that need to be present in the phrase.
phrase String Phrase to search in to.

isObject(value)

Checks if given value is an object. NOTE: null is not treated as an object, this is different that typeof operator.

Usage

    import { isObject } from "utility-belt";
    ...

    console.log(
      isObject(\\{}\\),            -\// \\true\\ \-
      isObject(true),          -\// false\-
      isObject("object"),      -\// false\-
      isObject([]),            -\// false\-
      isObject(1),             -\// false\-
      isObject(null),          -\// false\-
      isObject(Symbol()),      -\// false\-
      isObject(function(){ }), -\// false\-
      isObject(undefined),     -\// false\-
    );

Props

Name Type Description
value Any Value to be check if it is an Object.

inPolygon(point, polygon)

Check if piont [x,y] is in polygon [[x,y],...].

Usage

    import { inPolygon } from "utility-belt";
    ...

    console.log(
      inPolygon(\\[-1, 1]\\, [\\[-1, 3], [-3, 0], [-2, 1], [3, 1], [1, 1], [1, 3]\\])

Props

point Array X and Y coordinates of point that wer're looking for.
polygon Array Array of arrays (points).

wordCase(options)(value)

Usage

    import { wordCase } from "utility-belt";
    ...

    const \\caseOptions\\ = {
      one: "zadanie",
      couple: "zadania",
      multiple: "zadań"
    };

    const wcase = wordCase(\\caseOptions\\);

    console.log(
      `Masz jedno nowe ${wcase(\\1\\)}.`,  -\// Masz jedno nowe \\zadanie\\.\-
      `Masz dwa nowe ${wcase(\\2\\)}.`,    -\// Masz dwa nowe \\zadania\\.\-
      `Masz pięć nowych ${wcase(\\5\\)}.`  -\// Masz pięć nowych \\zadań\\.\-
    );

Props

Name Type Description
options Object Configuration object that contains three word cases (for Polish language)
  • one - case for single element.
  • couple - case for 2, 3, 4 and all n*10 + 2, n*10 + 3, n*10 + 4 elements, where n starts from 2.
  • many - case for 11, 12, 13, 14, and all other numbers of elements not specified above.
value Number Numeric value that the word case should match.
