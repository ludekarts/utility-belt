deepClone(source)

Allows to deeply clone source object.


Usage

  import { deepClone } from "@ludekarts/utility-belt";
  ...

  const source = {
    user: {
      name: "Marry",
      age: 21,
      gender: "female",
      metadata: {
        auth: true,
      }
    }
  };

  const \\clone\\ = deepClone(source);

  console.log(source.user.metadata !== clone.user.metadata);  -\// true\-

Props

Name Type Description
source Any Instance that will be deeply cloned
