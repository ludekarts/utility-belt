events(root, event, ...)

Enables stright forward event delegation system in vanilla JS.

Usage

  import { events } from "utility-belt";
  ...

  const handlers = events(\\document.body\\, "\\click\\", "\\dblclick\\");

  handlers.add("\\actionAname\\", event => {
    -\// Handle action\-
  });

  handlers.callHandler("\\actionAname\\", {});

Now in HTML you need to declare which elementc can trigger events:

Props

Name Type Description
root HTMLElement Reference to the DOM element that will hanle event delegation
event String Name of event that will be handled
... String More event names
