PubSub(config)

Basic implementation of Publish/Subscribe pattern with namespaces.

Usage

  import { PubSub } from "utility-belt";
  ...

  const \\pubsub\\ = PubSub();

  \\pubsub\\.subscribe("myEvent", console.log);
  \\pubsub\\.publish("myEvent", { hello: "world" }) -\\\ { hello: "world" }\-

  \\pubsub\\.subscribe("myNamespace", "myEvent", console.log);
  \\pubsub\\.publish("myNamespace", "myEvent", { hello: "world" }) -\\\ { hello: "world" }\-

Props

Name Type Description
config object
  • debug - if set to true then method getNamespace will be active.
  • logger(message, data) - custom loging function.

Api description

Method Description
subscribe(namespace, subject, callback) Adds new subscribe subject to given namespace.
unsubscribe(namespace, subject, callback) Removes subscription from the notification list in given namespace.
publish(namespace, subject, message) Publish/broadcast any message for given subject in given namespace.
Namespace if omitted it is set by default to /
