debounce(callback, delay)

Does not allo to run callback fn before given delay elapse.

Usage

    import { debounce } from "utility-belt";
    ...

    const handleScroll = \\debounce\\(event => console.log("scroll"), \\300\\);

    document.addEventListener("scroll", handleScroll);

Props

Name Type Description
callback Function Function that will be called after given delay elapse.
delay Number Delay in miliseconds after which callback will be called.

throttle(callback, delay)

Runs callback fn with given delay.

Usage

    import { throttle } from "utility-belt";
    ...

    const handleScroll = \\throttle\\(event => console.log("scroll"), \\300\\);

    document.addEventListener("scroll", handleScroll);

Props

Name Type Description
callback Function Function that will be called after given delay elapse.
delay Number Delay between fn calls in miliseconds.

inDeltaTime(callback, delta)

Call callback function if it's called within delta time range.

Usage

    import { inDeltaTime } from "utility-belt";
    ...

    const handleFastKeypress = \\inDeltaTime\\(event => console.log("It was fast!"), \\300\\);

    document.addEventListener("keydown", handleFastKeypress);

Props

Name Type Description
callback Function Function that will be called if handler executes in delta time.
delay Number Time in miliseconds in which fn call must occure.
