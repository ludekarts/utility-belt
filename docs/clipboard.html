clipboard()(valueToCopy)

Create utility enables copying values into the clipboard. Copy function returns true if action succeded and false if action failed.

Usage

  import { clipboard } from "utility-belt";
  ...

  const copy = clipboard();

  if (copy("\\Put this text in clipboard\\")) {
    alert("Copy succeded");
  } else {
    alert("Copy failed");
  }

Props

Name Type Description
valueToCopy String Value that will be copied to the clipboard.
