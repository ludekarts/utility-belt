highlight(text, phrases)

Wrap parts of text with <strong> tags that matches with given phrases.

Usage

  import { highlight } from "utility-belt";
  ...

  const result = highlight("Good and goofy", "\\go\\, \\fy\\");
  -\// <strong>\\Go\\</strong>od and <strong>\\go\\</strong>o<strong>\\fy\\</strong> \-

Props

Name Type Description
text String Text that contains words to be highlighted.
phrases String Coma-separated phrases to highlight.
