loopstack(length)

Provides you an array-like structure that can store only certain amount of data. If given length is exceaded loopstck starts to overrite data form the begining. This may be handy for implememnting simple undo functionality where system can remmember for example only last 10 actions.

Usage

  import { loopstack } from "utility-belt";
  ...

  const \\history\\ = loopstack(5);

  \\history\\.push("Apple");

  \\history\\.push("Banana");

  \\history\\.pull(); -\// Banana\-

  \\history\\.get(0); -\// Apple\-

Props

Name Type Description
length Number Size of the data structure.

Api description

Method Description
push(value) Pushes value into the data structure.
pull() Pulls last value from the data structure. Value is removed from the structure.
get(index) Get value from the structure at given index. Value is NOT removed from the structure.
head() Get currrent HEAD value from the structure. Value is NOT removed from the structure.
dump() Returns whole data structure as an array.
