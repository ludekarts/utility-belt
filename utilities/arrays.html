arrayToObject(array, key)

Transfrom array of objects in to JS object, stores each entry under the key pulled from the object.

Usage

    import { arrayToObject } from "utility-belt";
    ...

    const \\array\\ = [{ name: "John", age: 22 }, { name: "Marry", age: 21 }];

    console.log(arrayToObject(array, "\\name\\"));
    -\/*
    {
       John: { name: "John", age: 22 },
       Marry: { name: "Marry", age: 21 }
    }
    */\-

reduce(array, reducer, init)

Custom implementation of Array.reduce() method. One significant change is that reduce function recieve fourth parmater that tells if current item is the last one. isLast flag can be usefull for example to perform some cleanups/wrapups at the last step of reduction.

Usage

    import { reduce } from "utility-belt";
    ...

    const reduced = reduce(
      array,
      (\\acc\\, \\current\\, \\index\\, \\isLast\\) => {},
      []
    );

sortByPhraseIndex(array, phrase, selector)

Sorts itmes in array according to apperance (index) of given phrase. Lower index is better a.k.a. closer to begin of string.

NOTE: This fn will only sort items that contains given phrase. Remaining elements will be left in order they're in the array, so in case you want them to be for example sorted alphabeticall, you need to do it by yourself (see usage below).

Usage

    import { sortByPhraseIndex } from "utility-belt";
    ...

    const \\array\\ = ["zero", "big", "deep", "deer", "bolder", "older"];

    array.sort(); -\// Sort array alphabetically\-

    console.log(
      sortByPhraseIndex(\\array\\, "\\de\\")
    );
    -\// ["\\de\\ep", "\\de\\er", "ol\\de\\r", "bol\\de\\r", "zero", "big"]\-

Props

Name Type Description
array Array Array to be processed.
phrase String Phrase that should apear in array items.
selector Function Function that allows to select filed for sorting if we're dealing with an object e.g. (item) => item.name
