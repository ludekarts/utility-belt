PubSub(config)

Basic implementation of Publish/Subscribe pattern with namespaces.

Usage

  import { PubSub } from "@ludekarts/utility-belt";
  ...

  const \\pubsub\\ = PubSub();

  \\pubsub\\.on("myEvent", console.log);
  \\pubsub\\.dispatch("myEvent", { hello: "world" }) -\\\ { hello: "world" }\-

  \\pubsub\\.on("myNamespace", "myEvent", console.log);
  \\pubsub\\.dispatch("myNamespace", "myEvent", { hello: "world" }) -\\\ { hello: "world" }\-

Props

Name Type Description
config object
  • debug - if set to true then method getNamespace will be active.
  • logger(message, data) - custom loging function.

Api description

Method Description
on(namespace, subject, callback) Adds new subscribe subject to given namespace.
off(namespace, subject, callback) Removes subscription from the notification list in given namespace.
dispatch(namespace, subject, message) Publish/broadcast any message for given subject in given namespace.

By default the namespace is set to "/"
