elFromString(HTMLString)

Creates HTML element from given string.

Usage

    import { elFromString } from "@ludekarts/utility-belt";
    ...

    document.body.appendChild(
      elFromString("
Hello
")
    );

Props

Name Type Description
HTMLString String HTML code in string format that will be transformed into document-fragment

getNodesPath(stopSelector)

Returns list of parent nodes

Usage

    ...

    document.body.appendChild(
      elFromString("
Hello
")
    );

Props

Name Type Description
